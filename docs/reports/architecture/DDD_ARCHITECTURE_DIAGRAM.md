## Key Architectural Improvements

### 1. **Dependency Inversion Principle (DIP)**

**Before:**
```typescript
// Application layer directly importing Infrastructure
import { WorkspaceRuntimeFactory } from '@infrastructure/runtime/...';
```

**After:**
```typescript
// Application layer depends on abstraction
import { WORKSPACE_RUNTIME_FACTORY } from '@application/tokens/...';

const factory = inject(WORKSPACE_RUNTIME_FACTORY);
```

### 2. **Interface Segregation**

**Before:**
```typescript
// Presentation directly using Domain types
@Input() eventBus?: WorkspaceEventBus;  // Domain type
implements Module  // Domain interface
```

**After:**
```typescript
// Presentation using Application interfaces
@Input() eventBus?: IModuleEventBus;  // Application type
implements IAppModule  // Application interface
```

### 3. **Adapter Pattern**

**Before:**
```typescript
// Direct Domain event bus usage
eventBus.publish(event);  // Domain WorkspaceEventBus
```

**After:**
```typescript
// Wrapped via adapter
const eventBus: IModuleEventBus = facade.getEventBus(workspaceId);
eventBus.publish(event);  // Application IModuleEventBus
```
